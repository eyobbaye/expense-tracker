💸 ExpenseFlow — Smart Personal Finance Tracker
ExpenseFlow is a sleek, responsive, and intuitive personal finance dashboard built with ReactJS and Tailwind CSS. It enables users to seamlessly log daily earnings and expenses, automatically calculate net balances, and visualize their financial habits through an elegant, modern user interface.

✨ Key Features
💵 Dual Ledger Tracking: Effortlessly add and categorize both Earnings and Expenses with automatic transaction timestamps.

🧮 Real-Time Calculations: Dynamic state management instantly deducts expenses from cumulative earnings to display your accurate, up-to-date net balance.

📊 Financial Dashboard: Get an immediate overview of your financial health with high-level summary cards showing Total Income, Total Expenses, and Net Balance.

📱 Fully Responsive Architecture: Crafted using Tailwind CSS's mobile-first responsive utilities, offering a seamless experience across desktops, tablets, and smartphones.

🚀 Tech Stack
Frontend Library: ReactJS (Functional Components & Hooks)

Styling Engine: Tailwind CSS (Utility-first framework)

Icons: Lucide React / React Icons (Clean, vector iconography)

Build Tool: Vite (Next-generation, blazing-fast frontend tooling)

🛠️ Installation & Setup
Follow these simple steps to get a local copy of ExpenseFlow up and running on your machine.

Prerequisites
Ensure you have Node.js (v14.x or higher) and npm installed.

1. Clone the Repository
Bash
git clone https://github.com/eyobbaye/expense-tracker.git
cd Expense-Tracker
2. Install Dependencies
Bash
npm install
3. Start the Development Server
Bash
npm run dev
Open your browser and navigate to the local server address (usually http://localhost:5173) to view the application.
